# Ziglang devcontainers template

This repo provides a devcontainer template for Ziglang development. 

## Supported Zig versions
* 0.13.0


## Included tools
* [Zig toolchain](https://ziglang.org/)
* [ZLS](https://zigtools.org/zls/install/)
